# 紫龍 Purple Dragon

A two-Michelin-star omakase sushi experience — landing page + full menu.
Built with **Next.js (App Router)** and **Tailwind CSS**. UI/UX & frontend showcase.

## Routes
- `/` — landing page (`app/page.jsx`)
- `/menu` — full filterable menu (`app/menu/page.jsx`)

## Develop
```bash
npm install
npm run dev      # http://localhost:3000
npm run build && npm start
```
